For CoogBay, a full-stack web application designed exclusively for University of Houston students to buy and sell products or services, I led the development using the Next.js App Router framework for the frontend and MySQL with Prisma ORM for the backend. I implemented user authentication and verification using Auth0 and Resend, integrated Stripe for payment processing, and designed a scalable schema with models for users, posts, roles, and verification tokens. I configured the project to use PlanetScaleDB for seamless MySQL database hosting and created seed scripts to populate initial data with securely hashed passwords. The application supports image uploads, responsive UI with Tailwind CSS, and features user profiles that display personal info, favorites, and optionally listed products or services. I also developed the backend API for full CRUD operations, structured the app with clean folder organization, and ensured smooth local development by optimizing the Prisma client setup.
