Developed a full-stack web application using Django with SQLite3 as the database for lightweight, efficient data storage.  
Designed the frontend using HTML, CSS, and Bootstrap to ensure a responsive and visually appealing user interface.  
Implemented Django’s MVT (Model-View-Template) architecture to manage dynamic content and streamline backend logic.  
Utilized Django ORM for seamless interaction with the SQLite database, and incorporated Django's built-in authentication system for secure user login and registration.  
Customized the Django admin panel for efficient backend management and leveraged template inheritance for consistent layout and reduced redundancy.
